# ASC-ODE
## Build
This repository depends on the following tools:
- [Just](https://github.com/casey/just)
- [uv](https://github.com/astral-sh/uv)
- [cmake](https://cmake.org/)

The repository can be build with `just`. For more information run:
```shell
just
```
